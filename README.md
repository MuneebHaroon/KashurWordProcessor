# KashurWordProcessor
A word processor with basic wordpad like abilities for Kashmiri / Urdu language. The word processor can allow you to create documents in Nastaliq script.
This word processor was created in C#. The documents created are in rtf format and can be opened by most word processors. You do not need a Kashmiri or Urdu keyboard at all.
The application uses .net frameworks 4.8. If you are using windows 11 or Windows 10 the chances are that you alreeday have this installed on your PC. If not, then you can download the runtime version directly from Microsoft at https://dotnet.microsoft.com/en-us/download/dotnet-framework/net48 .
